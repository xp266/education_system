# 教育系统智能体

一个基于大语言模型的教育管理系统智能助手，为学生和教师提供智能问答、政策查询、数据查询等服务。

## 项目简介

本项目是一个多智能体协作的教育系统智能助手，集成了意图分析、RAG知识库检索、智能数据库查询等功能，通过自然语言交互，为教育场景提供智能化服务。

## 核心功能

### 1. 智能意图分析

- 自动识别用户查询意图，包括：
  - 复杂查询
  - 数据库查询
  - 学生手册/政策查询
  - 普通对话

### 2. RAG知识库检索

- 基于 bge-small-zh-v1.5 向量化模型
- 学生手册文档向量化存储
- 语义检索相关政策内容
- 智能总结检索结果

### 3. 智能数据库查询

- 自然语言转SQL查询
- 学生信息查询
- 成绩查询
- 课程信息查询
- 教师信息查询

### 4. 多智能体协作

- 意图分析智能体
- 对话生成智能体
- 数据库查询智能体
- 政策查询智能体
- 复杂查询协调智能体

### 5. Token追踪与管理

- 实时统计Token消耗
- 多阶段Token统计
- 详细Token报告

## 技术架构

### 后端技术栈

- \*\*Web框架：Flask
- **数据库**：MySQL
- **AI模型**：适配各种大模型
- **向量化模型**：bge-small-zh-v1.5
- **向量检索**：sentence-transformers

### 前端技术栈

- **Web框架**：Flask (前端服务)
- **UI**：原生HTML/CSS/JavaScript
- **Markdown渲染**：marked.js
- **数学公式**：KaTeX

### 核心模块

```
AI_Tool/
├── intent/           # 意图分析模块
├── llm/            # 大语言模型调用模块
├── prompts/        # 提示词管理模块
├── tools/          # 工具模块
```

## 快速开始

### 环境要求

- Python 3.8+
- MySQL 8.0+

### 安装步骤

1. **克隆项目**

```bash
git clone <repository-url>
cd 教育系统智能体
```

1. **安装依赖**

```bash
pip install -r requirements.txt
```

1. **配置环境变量**

```bash
cp .env.example .env
# 编辑 .env 文件，填入你的配置
```

1. **初始化数据库**

```bash
cd database
python create_sql.py
```

1. **启动服务**

启动后端服务（端口5001）：

```bash
cd backend
python app.py
```

启动前端服务（端口5000）：

```bash
cd frontend
python app.py
```

1. **访问应用**

打开浏览器访问：`http://localhost:5000`

## 项目亮点

1. **多智能体协作**：多个专门的智能体分工协作，处理不同类型的查询
2. **RAG知识库检索**：基于语义检索学生手册，提供精准的政策咨询
